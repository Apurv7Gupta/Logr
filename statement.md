# Project Statement – Student Habit Tracking System

## 📌 Problem Statement

Students often struggle to maintain consistent daily habits such as studying, reading, or exercising.  
A simple system is required to add habits, track progress, and view performance reports.

---

## 🎯 Scope of the Project

This project includes:

- Habit creation and deletion
- Daily progress tracking
- Generating summary reports
- Console-based UI
- Modular Java implementation

---

## 👥 Target Users

- Students who want to track habits
- Individuals who prefer a simple, lightweight tracking tool
- Beginners learning Java and OOP concepts

---

## 📝 High-Level Features

1. _Habit Module_
   - Add habit
   - Delete habit
   - View habit list

2. _Progress Module_
   - Mark habit as completed
   - Count total completed days

3. _Report Module_
   - Display habit summary
   - Show progress reports
