# Logr

---

📌 Project Description

This project is a Java-based habit tracking system that enables students to maintain and monitor their daily habits. It provides essential features such as habit management, progress tracking, and report generation. The application follows Object-Oriented Programming (OOP) principles and is executed using the Command Prompt (CMD).
The main objective of this project is to help students stay consistent with their daily routines like studying, reading, exercising, etc., by offering a simple and efficient habit tracking tool.

---

🎯 Objectives

To implement a console-based habit tracking system.

To demonstrate OOP concepts such as classes, objects, encapsulation, and modularity.

To allow students to add, delete, view, and track habits.

To generate habit progress summaries.

---

🧩 Functional Features

1. Habit Management

Add Habit

Delete Habit

View All Habits

2. Progress Tracking

Mark habit as completed for the day

3. Reporting Module

Display habit summary

Show total completed days

---

⚙ Non-Functional Features

Usability: Easy-to-understand text-based interface

Maintainability: Modular code using separate Java classes

Performance: Quick response using in-memory lists

Scalability: Can be extended to GUI or database version
